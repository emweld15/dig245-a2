Viewing HTML in Chrome on Macbook

"Going through dirty laundry"

We will be viewing

  • Open up your webpage in the Chrome search bar.
  • Press "View" at the top bar of you screen.
  • From that drop down, select "Developer" and then "View Source".
  • After selecting "View Source" you should see the HTML code appear on your screen Vestique homepage!
  • If you wish to stop viewing the HTML code on the Vestique homepage, simply delete "view-source" from the URL in the search bar.
