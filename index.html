Viewing HTML in Chrome on Macbook

"Going through dirty laundry"

We will be viewing the clothing store Vestique and its homepage (to connect with my clothing and dirty laundry metaphor all the better!)

  1. Open up your webpage in the Chrome search bar.
  2. Press "View" at the top bar of you screen.
  3. From that drop down, select "Developer" and then "View Source".
  4. After selecting "View Source" you should see the HTML code appear on your screen Vestique homepage!
  5. If you wish to stop viewing the HTML code on the Vestique homepage, simply delete "view-source" from the URL in the search bar.
